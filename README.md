# 无极 ARK Infinity Server — 方舟服务器一体化管理解决方案

> 让服主专注运营，让玩家沉浸体验。

---

## 产品简介

**无极 ARK Infinity Server** 是一款面向《方舟：生存进化》（ASE）与《方舟：生存飞升》（ASA）服务器的**全功能管理与运营平台**。

它将服务器管理、玩家服务、商城运营、社群互动、小游戏娱乐等核心运营场景整合到一个统一的 WPF 桌面程序中，帮助服主实现**一站式服务器运营管理**。

---

## 核心功能一览

### 🏪 商城管理系统

- **可视化商品编辑** — 支持物品、恐龙、补给箱、经验、礼包、兑换码等多种商品类型的可视化配置
- **双货币体系** — 金币 + 钻石双货币系统，灵活定价策略
- **以物易物商店** — 支持玩家间道具交换功能
- **套装礼包** — 一键配置新手礼包、VIP礼包等组合商品
- **在线商城** — 内置 Web 商城，玩家可通过浏览器直接浏览购买
- **多服务器同步** — 一键将商城配置同步到多个游戏服务器

### 💰 在线充值系统

- **多支付平台对接** — 支持多种第三方支付平台接入
- **充值卡密管理** — 支持批量生成、激活、查询卡密
- **在线充值** — 玩家可通过群聊指令或网页自助充值
- **兑换码系统** — 支持活动兑换码发放与核销
- **充值记录追溯** — 完整的充值流水与审计日志

### 💬 社群聊天系统

- **QQ/Kook 群互通** — 游戏内聊天与 QQ 群、Kook 频道双向同步
- **跨服聊天** — 多地图服务器之间实时聊天同步
- **群内自助服务** — 玩家可通过群聊指令查询信息、转账、充值
- **敏感词过滤** — 内置敏感词过滤系统，维护社群环境
- **关键词自动回复** — 自定义关键词触发自动回复

### 🎮 群内小游戏系统

- **骰子比大小** — 经典骰子博弈，支持多种玩法
- **恐龙赛跑** — 恐龙竞速投注系统
- **大小竞猜** — 简单刺激的数字竞猜
- **农场种植** — 模拟农场经营玩法
- **钓鱼系统** — 钓鱼收集与竞赛
- **猜数字** — 经典猜数字游戏
- **彩票系统** — 多玩法彩票投注
- **挖矿系统** — 模拟挖矿收益玩法
- **抢夺系统** — 玩家间互动抢夺玩法
- **老虎机** — 经典老虎机博彩

### 🎁 玩家运营系统

- **签到奖励** — 每日签到激励玩家活跃
- **排行榜** — 多维度排行榜展示（金币、等级等）
- **邀请返利** — 老玩家邀请新玩家获得奖励
- **红包系统** — 群内红包发放与抢红包
- **邮件系统** — 支持批量发送邮件礼包到玩家游戏邮箱
- **VIP 体系** — 支持永久 VIP 与限时 VIP，多等级权限组
- **积分兑换** — 在线时长兑换奖励

### 🏛️ 拍卖行系统

- **玩家自由交易** — 玩家可在群内发布拍卖物品
- **竞价机制** — 支持多人竞价，价高者得
- **冻结管理** — 交易资金与物品安全冻结
- **历史记录** — 完整的拍卖历史可追溯

### 🤖 AI 智能助手

- **多 AI 模型接入** — 支持 OpenAI、DeepSeek、通义千问、智谱、豆包、Google Gemini、Ollama 本地等多种 AI 提供商
- **知识库系统** — 可自定义知识库，回答服务器规则、玩法说明等
- **联网搜索** — AI 可联网搜索，提供更丰富的回答
- **群内智能回复** — AI 自动参与群聊，解答玩家问题

### 🔌 插件统一管理

- **插件配置可视化** — 可视化编辑各插件配置文件
- **恐龙授权系统** — 精细控制恐龙携带权限
- **物品授权系统** — 控制特定物品的使用权限
- **事件倍率配置** — 灵活调整游戏内事件倍率
- **聊天展示配置** — 自定义聊天消息展示样式
- **插件热部署** — 支持配置远程下发至游戏服务器

### 🛡️ 服务器管理

- **多服务器管理** — 同时管理多个游戏服务器实例
- **RCON 远程控制** — 通过 RCON 协议远程执行服务器命令
- **玩家管理** — 查看在线玩家、封禁/解封玩家
- **远程重启/关机** — 群内管理员可远程重启或关闭服务器
- **崩溃自动恢复** — 服务器崩溃后自动保存并重启
- **远程恐龙清理** — 清理野生或模组恐龙

### 📊 数据管理

- **数据库管理** — 支持 MySQL 与 SQLite 数据库管理
- **数据备份** — 定时自动备份数据库
- **数据查询** — 可视化查询玩家数据、交易记录等
- **表结构查看** — 直观查看数据库表结构与索引信息

### 🔐 授权管理系统

- **机器码授权** — 基于机器码的软件授权验证
- **卡密充值** — 通过卡密为授权续期
- **RSA 签名验证** — 安全的签名验证机制
- **授权状态管理** — 支持永久授权与限时授权

## 技术优势

| 优势 | 说明 |
|------|------|
| **跨游戏支持** | 同时支持 ASE（生存进化）与 ASA（生存飞升）两个版本 |
| **高性能架构** | 基于 .NET 8 + WPF，MVVM 架构，万行配置数据流畅编辑 |
| **模块化设计** | 各功能模块独立，可按需启用，灵活扩展 |
| **可视化配置** | 所有配置项均提供可视化界面，无需手动编辑 JSON |
| **安全可靠** | RSA 签名验证、敏感词过滤、操作审计等多重安全保障 |
| **自动更新** | 内置更新器，自动检测并安装最新版本 |

## 适用场景

- **私人服务器** — 个人或小团队运营的方舟私服
- **商业服务器** — 面向公众收费运营的方舟服务器
- **多服矩阵** — 同时管理多个地图/多个版本的服务器集群
- **社群运营** — 需要 QQ 群/Kook 社群与游戏深度联动的场景

## 系统要求

- **操作系统**：Windows 10/11（64位）
- **运行环境**：.NET 8.0 Runtime
- **数据库**：SQLite（内置）/ MySQL（可选）
- **网络**：服务器需开放 WebSocket 通信端口
